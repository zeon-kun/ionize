# Vault — Map of Content

Reusable daily-workflow vault: Obsidian knowledge base + the `ionize` knowledge-graph pipeline.

## Conventions
- [[CONTRIBUTING-DOCS]] — living-doc template + doc-sync rule.
- Templates: `templates/living-doc.md`, `templates/node.md`.

## Features
- [[ionize]] — 3-pass knowledge-graph extraction → Obsidian Canvas. Pass 1 (code) working.

## Generated
- `graph/graph.canvas` — visual graph (open in Canvas).
- `graph/nodes/*.md` — per-node notes (rendered by the native Graph View).
